# 🛠️ Web Scraper – Automated Data Extraction Tool  

## 🚀 Overview  
This project is a **web scraper** that extracts structured data (like product listings) from websites and saves it in CSV format.

## 📦 Features  
- ✅ Extracts product **title, price, and link**  
- ✅ Saves data to **CSV for easy analysis**  
- ✅ Uses `BeautifulSoup` and `requests`  
- ✅ Can be modified for **different websites**
